# Mapa colaborativo (web app / PWA) - opção B (mapa grátis)

Você pediu um app web que:
- abre um mapa "tipo Google Maps" (arrastar, zoom, mundo todo)
- pega GPS exato (quando permitido) OU permite marcar manualmente no mapa
- usuários registram roubos/furtos/tentativas com observação
- todo mundo vê pontos + mapa de calor

Este projeto faz isso usando:
- Mapa grátis: OpenStreetMap + Leaflet
- Banco compartilhado (para todos verem): Supabase (bem simples para começar)

---

## 1) O que você precisa criar no Supabase (em 5 minutos)

1. Crie uma conta no Supabase e um projeto.
2. Vá em **SQL Editor** e rode este SQL:

```sql
create table if not exists public.incidents (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default now(),
  occurred_at timestamp with time zone not null,
  type text not null,
  note text,
  lat double precision not null,
  lng double precision not null
);
```

3. Vá em **Table editor** e confirme que a tabela `incidents` existe.

---

## 2) Liberar leitura/escrita (para o MVP)

Para um protótipo, você pode liberar leitura/escrita públicas com RLS (políticas).
No Supabase, vá em **Authentication -> Policies** (ou Table -> RLS) e:

1) Ative RLS na tabela `incidents`
2) Crie duas policies simples:

- SELECT: allow public read
- INSERT: allow public insert

Exemplo (SQL) se preferir:

```sql
alter table public.incidents enable row level security;

create policy "public read"
on public.incidents for select
to anon
using (true);

create policy "public insert"
on public.incidents for insert
to anon
with check (true);
```

⚠️ Atenção: isso é para MVP. Depois dá para colocar regras anti-spam (limites, login, etc).

---

## 3) Colar as chaves no app

Abra `app.js` e cole:

- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`

Você encontra isso em Supabase -> **Project Settings -> API**.

---

## 4) Rodar no seu computador (sem complicação)

O jeito mais fácil:
- Instale o **VS Code**
- Instale a extensão **Live Server**
- Abra a pasta do projeto e clique em "Go Live"

Ou, se você tiver Python instalado:

```bash
python -m http.server 8000
```

Depois abra:
http://localhost:8000

## 6) Privacidade (ponto “aproximado”)

O app salva o ponto exato no banco, mas pode mostrar levemente deslocado no mapa público.
Isso é controlado por:

`DISPLAY_JITTER_DEGREES` no `app.js` (padrão ~50m).
